# Events & Concerts of Bangladesh (ECB)

A community-driven open-source database and resource hub for concerts, festivals, artists, venues, and event promotion in Bangladesh.

## Features

- Concert database
- Artist database
- Venue database
- AI poster prompt library
- Event resources

## Contributing

Anyone can submit:

- New concert information
- Artist profiles
- Venue information
- Concert poster prompts
